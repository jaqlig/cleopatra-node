# Cleopatra
### Backend for https://github.com/dereniowskid885/cleopatra-front-end
---
## Prerequisite
- Node.js 16.x.x installed

## How to
    git clone https://github.com/jaqlig/cleopatra-node.git
    cd cleopatra-node
    npm install
    DEBUG=cleopatra:* npm start

And go to http://localhost:3000/
